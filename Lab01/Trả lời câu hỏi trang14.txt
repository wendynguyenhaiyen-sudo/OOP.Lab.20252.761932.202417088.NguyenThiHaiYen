Câu hỏi 1: Điều gì xảy ra nếu người dùng chọn “Cancel”?

Khi người dùng chọn Cancel: JOptionPane.showConfirmDialog trả về giá trị JOptionPane.CANCEL_OPTION.
Trong đoạn code hiện tại, mọi lựa chọn không phải Yes đều được hiển thị là "No".
Về thực tế: đây là hành vi mặc định, nhưng nếu muốn phân biệt Cancel, bạn cần kiểm tra riêng CANCEL_OPTION và xử lý thông báo tương ứng.

+) Ví dụ rõ ràng:
	if(option == JOptionPane.YES_OPTION) {
	    JOptionPane.showMessageDialog(null, "You've chosen: Yes");
	} else if(option == JOptionPane.NO_OPTION) {
	    JOptionPane.showMessageDialog(null, "You've chosen: No");
	} else {
	    JOptionPane.showMessageDialog(null, "You've chosen: Cancel");
	}

Câu hỏi 2: Làm thế nào để tùy chỉnh các tùy chọn cho người dùng, ví dụ chỉ hai tùy chọn “Yes” và “No”, hoặc “I do” và “I don’t”?

Để tùy chỉnh nhãn nút, bạn dùng JOptionPane.showOptionDialog thay vì showConfirmDialog.
Ví dụ với hai tùy chọn "I do" và "I don't":
	Object[] options = {"I do", "I don't"};
	int option = JOptionPane.showOptionDialog(null,
	        "Do you want to join the event?",
	        "Choose your option",
	        JOptionPane.YES_NO_OPTION,
	        JOptionPane.QUESTION_MESSAGE,
	        null,
	        options,
	        options[0]);
	if(option == 0){
	    JOptionPane.showMessageDialog(null, "You chose: I do");
	} else if(option == 1){
	    JOptionPane.showMessageDialog(null, "You chose: I don't");
	} else {
	    JOptionPane.showMessageDialog(null, "You closed the dialog");
	}
Giải thích:
	+) options là mảng các nhãn nút muốn hiển thị.
	+) option trả về chỉ số của nút mà người dùng nhấn:
		0 → nút đầu tiên ("I do")
		1 → nút thứ hai ("I don't")
		-1 → người dùng đóng hộp thoại mà không nhấn nút nào
	=> Có thể tùy chỉnh bất kỳ nhãn nút nào và xử lý đúng hành động người dùng.
